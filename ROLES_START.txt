Роли пользователей
==================

После запуска приложения откройте http://localhost:8080 и войдите одним из тестовых аккаунтов:

ADMIN:
  login: admin
  password: admin123

OPERATOR:
  login: operator
  password: operator123

OBSERVER:
  login: observer
  password: observer123

ADMIN видит дополнительный раздел «Админ».
В нём можно:
- просматривать пользователей;
- менять роли ADMIN / OPERATOR / OBSERVER;
- блокировать и разблокировать пользователей;
- задавать новый пароль;
- смотреть журнал действий;
- отмечать действия как проверенные и добавлять комментарий администратора.

OPERATOR может управлять БПЛА и миссиями.
OBSERVER может только просматривать данные.
